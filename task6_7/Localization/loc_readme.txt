To update lang (.ts files) from program itself ('tr()' func or trasnaltable text in .ui):
> lupdate task6.pro

To get use-ready files (.qm) from .ts:
> lrelease task6.pro

If you have lupdate-qt6/lrelease-qt6 it is better to rename them to remove '-qt6'.

Files .qm automatically embedded via Recources.qrc
